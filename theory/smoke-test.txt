SMOKE TEST (First)
Purpose: Quick sanity check - does it work at all?
A shakeout test, sometimes known as a smoke test, is a small test that checks for major issues before spending more time and resources. It typically uses one or a few VUs that run for a short amount of time

import http from 'k6/http';

    duration: '30s',  // Very short duration
    thresholds: {
        http_req_duration: ['p(95)<1000'],  // Basic threshold
        http_req_failed: ['rate<0.1'],
    }
};

export default function () {
    http.get('https://your-api.com/endpoint');
}
